# Cinematique - 150 Cinematic Techniques for Obsidian

An Obsidian vault with 150 cinematic techniques from [vvsvs.pro/cinematique](https://vvsvs.pro/cinematique), organized as a browsable reference library for AI image/video prompting.

## What's Inside

- **150 technique notes** covering Camera Work, Lighting, Composition, Editing, Storytelling, VFX, and Genres & Styles
- **191 media files** - cover images (PNG/JPG) + demo videos (MP4) for Camera Work techniques
- **Obsidian Bases view** (`Cinematique.base`) with gallery, table, and per-category card views

## Structure

```
cinematique/
├── Cinematique.base       # Obsidian Bases view (gallery + 13 views)
├── notes/                 # 150 technique markdown files
│   ├── Aerial Shot.md
│   ├── Film Noir.md
│   └── ...
└── attachments/           # Cover images + demo videos
    ├── aerial-shot.png
    ├── aerial-shot.mp4
    └── ...
```

## Usage in Obsidian

1. Clone or download this repository
2. Open Obsidian and add the `cinematique/` folder as a vault
3. Open `Cinematique.base` to browse techniques by category, difficulty, or mood

Requires **Obsidian 1.9+** with Bases support enabled.

## Note Format

Each technique note includes:

- Frontmatter: `category`, `difficulty`, `mood`, `has_video`, `cover_file`, tags
- Definition and cinematic history
- When to Use
- Best Practices
- AI Prompt Template (works with any image/video generator)
