### EXACT COLOR PALETTE (from competitor analysis)

```
PRIMARY COLORS:
- Background:       #1A1A1F (deep charcoal/night) or #0E0E12 (near-black)
- Primary text:     #E8D9A8 (parchment cream)
- Accent text:      #D4AF37 (gold leaf)
- Subheader text:   #C0392B (vermilion red)

FIGURE COLORS:
- Armor/mail:       Iron grey #6E7378, polished steel highlights
- Robes/cloaks:     Vermilion #C0392B, Ultramarine #284B8C, parchment #E8D9A8
- Skin tones:       Warm European Caucasian, weathered
- Heraldry/gold:    Gold leaf #D4AF37 for crowns, crosses, trim

SUPPORTING COLORS:
- Iron grey:        #6E7378 (armor/stone/shadows)
- Ultramarine:      #284B8C (sky/banner accent)
- Blood/fire:       #8B1E1E (battle accent, torchlight)
- Stone brown:      #5A4B3A (castle walls/ground)
```

### TYPOGRAPHY SPECS (CRITICAL — Copy đối thủ chính xác)

```
PRIMARY FONT (Main title):
- Name: BLACKLETTER / GOTHIC DISPLAY (e.g. UnifrakturCook, Cloister Black)
- Alternative: A strong chiseled medieval serif (Trajan Pro / Cinzel Bold)
- Style: Serif, carved-stone or illuminated-manuscript feel
- Weight: Bold or Extra Bold (700-900)
- Case: ALL CAPS
- Color: Parchment cream #E8D9A8
- Size: ~120-180pt for primary words

SECONDARY FONT (Subheader like "EXPLAINED"):
- Same font family as primary (chiseled serif preferred for legibility)
- Lighter weight (500-600)
- Smaller size: ~70-100pt
- Color: Gold leaf #D4AF37
- Often italic or regular
- Case: ALL CAPS or Title Case

TERTIARY FONT (Small details like dates/places):
- Same family
- Regular weight (400)
- Size: ~30-50pt
- Color: Parchment cream #E8D9A8 or vermilion
- Used for: Map labels, kingdom names, year dates, sub-explanations
```

## 📐 THE 6 PROVEN PATTERNS (From Competitor Analysis)

### 🅰️ PATTERN A: "EVERY X EXPLAINED" (40% of viral thumbs)

```
LAYOUT:
┌─────────────────────────────────────────┐
│         EVERY [TOPIC]                   │  ← Top 30% — Big serif
│           EXPLAINED                     │  ← Subheader smaller
│                                         │
│  [Oil painting visual centered/below]   │  ← Main figure/object
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title starts with "Every X"
- Topic is figure/object/concept collection
- Examples: "Every Crusade", "Every Medieval King", "Every Knightly Order"

VISUAL ELEMENTS:
- Single figure (a king, a knight, etc.) prominently
- OR multi-element (map with campaign arrows, banners)
- OR icon grid

PROVEN EXAMPLES:
- "EVERY CRUSADE EXPLAINED" + crusader knight + Holy Land map arrows
- "EVERY ENGLISH KING EXPLAINED" + 9 crown icons
- "EVERY KNIGHTLY ORDER explained" + heraldic shield grid
```

### 🅱️ PATTERN B: "TITLE STATEMENT" (25%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│      [BIG BOLD STATEMENT]               │  ← Top 30-40%
│                                         │
│  [Oil painting scene/comparison below]  │  ← Visual telling story
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title is question or statement (not "Every X")
- Concept needs visual storytelling
- Examples: "How the Black Death Killed Half of Europe", "Why Castles Stopped Being Built"

VISUAL ELEMENTS:
- Multi-panel comparison (before/after, siege then/now)
- Or storytelling scene
- Or figure + setting

PROVEN EXAMPLES:
- "HOW THE BLACK DEATH KILLED EUROPE" + 4-panel plague timeline
- "WHY KNIGHTS WORE HEAVY ARMOR" + armored knight + longbow
- "WAS THE MEDIEVAL ERA REALLY DARK? THE DARK AGES EXPLAINED" + monk in scriptorium
```

### 🅲️ PATTERN C: "CONQUEROR VS CONQUERED" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│  CONQUEROR             |  CONQUERED      │  ← Top — 2 words split
│                                         │
│  [Left scene]          |  [Right scene] │  ← Visual contrast
│                                         │
│  [BOTTOM LABEL/TAGLINE]                 │  ← Caption
└─────────────────────────────────────────┘

USE WHEN:
- Title implies contrast/comparison between two sides
- Two opposing rulers, armies, or outcomes
- Examples: "Normans vs Saxons", "The Winner and Loser of Hastings"

VISUAL ELEMENTS:
- Split composition (vertical or diagonal)
- Two contrasting figures or armies (victor crowned, vanquished fallen)
- Optional bottom caption box

PROVEN EXAMPLES:
- "CONQUEROR | CONQUERED" + William the Conqueror vs King Harold
- "THE CITY THAT WON AND THE CITY THAT FELL" + 2 besieged castles
```

### 🅳️ PATTERN D: "RULER GRID" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│       EVERY [RULER GROUP]               │  ← Header
│           EXPLAINED                     │
│                                         │
│   [grid of 6-12 ruler portraits]       │  ← Multi-figure grid
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Topic about multiple rulers/dynasties/figures
- Examples: "Every Plantagenet King", "Every Holy Roman Emperor", "Every Pope of the Crusades"

VISUAL ELEMENTS:
- 6-12 portraits in grid (2×3, 3×4, etc.)
- Each portrait in same oil painting style
- Subtle differences (age, crown, heraldry, regalia)
```

### 🅴️ PATTERN E: "QUOTE/CONCEPT BLOCK" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│   [SHORT QUOTE OR CONCEPT]              │  ← Top
│   [HIGHLIGHT WORD IN GOLD]              │
│                                         │
│  [Oil painting scene supporting concept]│
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title is statement/quote
- Counterintuitive concept
- Examples: "Knights Were Not Always Noble", "A Castle Was a House, Not a Fortress"

VISUAL ELEMENTS:
- Storytelling oil painting scene
- Often a single dramatic moment (a coronation, an oath, a betrayal)
- Figures in charged historical situations

PROVEN EXAMPLES:
- "MOST KNIGHTS COULD NOT AFFORD A HORSE" + dismounted knight in mud
- "THE POPE ONCE HAD THREE RIVALS" + three crowned clerics
```

### 🅵️ PATTERN F: "ICON ROW WITH LABELS" (5%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│      [TOPIC TITLE]                      │  ← Top
│                                         │
│   [Icon 1] → [Icon 2] → [Icon 3]        │  ← Process/progression
│   Label    Label       Label            │  ← Concept words
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title about progression/sequence
- Concept-driven, educational
- Examples: "The Rise of a Knight" (Page → Squire → Knight)

VISUAL ELEMENTS:
- 3-5 medieval icons/scenes in row (crown, sword, cross, castle, plague mask, shield)
- Arrows or dashes connecting
- Text labels (often with Latin or Old French terms)
- Sometimes period-language transliteration

PROVEN EXAMPLES:
- "THE MAKING OF A KNIGHT" + page → squire → dubbing → knight
- "HOW A SIEGE WORKED" + trebuchet → breach → assault → surrender
```
