```

TERTIARY FONT (Small details like locations):
- Same family
- Regular weight (400)
- Size: ~30-50pt
- Color: Marble white #E8E4D8 or Aegean blue
- Used for: Place names, epithets, dates, sub-explanations
```

## 📐 THE 6 PROVEN PATTERNS (From Competitor Analysis)

### 🅰️ PATTERN A: "EVERY X EXPLAINED" (40% of viral thumbs)

```
LAYOUT:
┌─────────────────────────────────────────┐
│         EVERY [TOPIC]                   │  ← Top 30% — Big serif
│           EXPLAINED                     │  ← Subheader smaller
│                                         │
│  [Oil-painted figure centered/below]    │  ← Main god/hero/monster
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title starts with "Every X"
- Topic is god/monster/labor collection
- Examples: "Every Labor of Hercules", "Every Olympian", "Every Titan"

VISUAL ELEMENTS:
- Single figure (Zeus, Hercules, Medusa, etc.) prominently
- OR multi-element (map of Greece, constellations)
- OR icon grid of mythic symbols

PROVEN EXAMPLES:
- "EVERY LABOR OF HERCULES EXPLAINED" + Hercules + Nemean lion
- "EVERY TITAN OF GREEK MYTH EXPLAINED" + 12 colossal figures
- "EVERY OLYMPIAN GOD explained" + portrait grid of Olympus
```

### 🅱️ PATTERN B: "TITLE STATEMENT" (25%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│      [BIG BOLD STATEMENT]               │  ← Top 30-40%
│                                         │
│  [Oil-painted scene/comparison below]   │  ← Visual telling story
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title is question or statement (not "Every X")
- Concept needs visual storytelling
- Examples: "How Pandora Doomed Mankind", "Why Zeus Feared the Fates"

VISUAL ELEMENTS:
- Multi-panel comparison (before/after, mortal/god)
- Or storytelling scene
- Or character + setting

PROVEN EXAMPLES:
- "HOW PANDORA DOOMED MANKIND" + 4-panel timeline of the jar opening
- "WHY ZEUS FEARED THE FATES" + Zeus + the three Moirai
- "WHO REALLY WON THE TROJAN WAR? EXPLAINED" + warrior at burning Troy
```

### 🅲️ PATTERN C: "HERO VS MONSTER" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│  HERO                  |  MONSTER       │  ← Top — 2 words split
│                                         │
│  [Left figure]         |  [Right figure]│  ← Visual confrontation
│                                         │
│  [BOTTOM LABEL/TAGLINE]                 │  ← Caption
└─────────────────────────────────────────┘

USE WHEN:
- Title implies a duel/confrontation
- Two opposing forces of a myth
- Examples: "Perseus vs Medusa", "Theseus and the Minotaur"

VISUAL ELEMENTS:
- Split composition (vertical or diagonal)
- Hero on one side, monster on the other
- Optional bottom caption box

PROVEN EXAMPLES:
- "PERSEUS | MEDUSA" + shielded hero vs serpent-haired Gorgon
- "THESEUS FACED THE MINOTAUR" + 2 figures at the labyrinth gate
```

### 🅳️ PATTERN D: "PANTHEON GRID" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│       EVERY [DEITY GROUP]               │  ← Header
│           EXPLAINED                     │
│                                         │
│   [grid of 6-12 god/hero portraits]    │  ← Multi-figure grid
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Topic about multiple gods/heroes/monsters
- Examples: "Every Olympian", "Every Greek Hero", "Every Primordial God"

VISUAL ELEMENTS:
- 6-12 oil-painted portraits in grid (2×3, 3×4, etc.)
- Each portrait in same neoclassical style
- Distinct attributes (Zeus's bolt, Poseidon's trident, Athena's helm)
```

### 🅴️ PATTERN E: "QUOTE/CONCEPT BLOCK" (10%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│   [SHORT QUOTE OR CONCEPT]              │  ← Top
│   [HIGHLIGHT WORD IN GOLD]              │
│                                         │
│  [Oil-painted scene supporting concept] │
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title is statement/quote
- Counterintuitive or fateful concept
- Examples: "No Mortal Escapes Their Fate"

VISUAL ELEMENTS:
- Storytelling oil-painted scene
- Often a single tragic or dramatic moment
- Figures caught in a moment of destiny or hubris
```

### 🅵️ PATTERN F: "ICON ROW WITH LABELS" (5%)

```
LAYOUT:
┌─────────────────────────────────────────┐
│      [TOPIC TITLE]                      │  ← Top
│                                         │
│   [Icon 1] → [Icon 2] → [Icon 3]        │  ← Process/progression
│   Label    Label       Label            │  ← Concept words
│                                         │
└─────────────────────────────────────────┘

USE WHEN:
- Title about progression/sequence
- Concept-driven, educational
- Examples: "The Ages of Man" (Gold → Silver → Bronze → Iron)

VISUAL ELEMENTS:
- 3-5 oil-painted mythic symbols/scenes in row
- Arrows or dashes connecting
- Text labels (often with Greek words)
- Sometimes original-language transliteration
```

## 🚨 FAILURE MODES TO AVOID

1. **Wrong Style** — Thumbnail MUST be neoclassical oil painting, NOT cartoon flat
2. **Wrong Background** — Dark charcoal with rim light, NOT bright flat full-frame
3. **Wrong Font** — Cinzel/Trajan SERIF, NOT Impact sans-serif
4. **Skip Pattern Selection** — Must analyze script and pick 3 distinct patterns
5. **Only 1 Version** — Must output BOTH with-text AND no-text for each variation
6. **Missing Canva Specs** — No-text version MUST include detailed Canva instructions
7. **Wrong Color** — Use exact hex codes (#E8E4D8 marble white, #C9A227 gold)
8. **Cartoon Slip** — Always emphasize "oil painting + chiaroscuro + brushwork"
9. **Full-Frame Visual** — Must leave dark space for text (top 35-40%)
10. **Generic Output** — Must reference specific gods/heroes/moments from script
